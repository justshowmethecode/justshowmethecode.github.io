hexo基本配置

安装并在本地运行

搭建过程需要安装git客户端,npm,nodejs可参考官网-地址

1.初始化hexo博客命令

hexo init blog      #初始化blog
cd blog
cnpm install
hexo g
hexo s

2.初始化完成后访问http://localhost:4000/

部署到github中

1.修改更路径下_config.yml

deploy:
  type: git
  repository: git@github.com:justshowmethecode/justshowmethecode.github.io.git
  branch: master

2.安装deploy插件

cnpm install hexo-deployer-git --save

3.部署

hexo d

4.访问
https://justshowmethecode.github.io/

icarus主题配置

官网

地址

安装

1.安装

git clone https://github.com/ppoffice/hexo-theme-icarus themes/icarus

2.配置根目录_config.yml

theme: icarus

3.生成icarus配置

hexo g
hexo s

新建基本页面

1.tags页面

hexo new page "tags"

2.分类主页

hexo new page "categories"

修改主页菜单

1.主题配置文件themes/icarus/_config.yml

navbar:
    # Navigation bar menu links
    menu:
        主页: /
        #归档: /archives
        分类: /categories
        标签: /tags
        #关于: /about

接入评论

1.主题配置文件themes/icarus/_config.yml

comment:
    type: valine
    app_id: DmzLuD1Do0eJ156ihBCtLvh0-gzGzoHsz        # (required) LeanCloud application id
    app_key: lxddgMoHBtoznGozilj8rtSS       # (required) LeanCloud application key
    notify: false           # (optional) receive email notification
    verify: false           # (optional) show verification code
    placeholder: '留下你的评论吧!'   # (optional) comment box placeholder text

中文转拼音

1.安装插件

cnpm i hexo-permalink-pinyin --save

2.配置根目录_config.yml

permalink_pinyin:
  enable: true
  separator: '-' # default: '-'
